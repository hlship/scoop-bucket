# hlship Scoop Bucket

A [Scoop](https://scoop.sh/) bucket for tools by [Howard Lewis Ship](https://github.com/hlship).

## Usage

```powershell
scoop bucket add hlship https://github.com/hlship/scoop-bucket
scoop install dexter
```

## Available Tools

| Tool | Description |
|---|---|
| [dexter](https://github.com/hlship/dexter) | Interactive dependency explorer for Clojure and JVM projects |
